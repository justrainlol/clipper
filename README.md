# Clipper

A lightweight, cross-platform media downloader GUI powered by yt-dlp. Built with Python and Tkinter, it allows you to instantly save videos, high-quality audio extractions, or animated GIFs from almost any link.

did i forget to say its open source too? yea, we ball like that.
## Features
- Multi-Platform Support: Native binaries for both Windows, Linux, and other operating systems.
- Format Toggles: Save files as Video (.mp4), Audio (.mp3), or animated GIFs.
- Quality Selectors: Choose explicit resolutions (up to 4K) and bitrates (up to 320kbps).
- Zero Configuration: Does everything for you, all you have to do is paste a link, bam!.
- Polished UI: Choose between Dark/Light mode, freindly UI.
- Lightweight: Extremely lightweight, only has what it needs, does not hog resources or run in the background.

## Installation and Running

### Windows
1. Go to the Releases tab on the right side of this repository.
2. Download the latest release of clipper.
3. Double-click the file to run.
4. > *Note for Windows users:* If you see a blue "Windows protected your PC" SmartScreen popup, click **"More info"** and then choose **"Run anyway"**. This warning appears simply because the executable is compiled via open-source automation and is not signed with an expensive commercial developer certificate.

### Linux
1. Go to the Releases tab.
2. Download the latest extensionless binary file (the one with no .exe).
3. Open your terminal, navigate to your downloads directory, and grant the binary execution rights:
   ```bash
   chmod +x clipper
   ```
